 Mattermost Scheduler Plugin

Плагин для Mattermost, который позволяет создавать и управлять запланированными сообщениями и напоминаниями.

- Команды
- Разработка
- Лицензия

 ✨ Возможности

- Запланированные сообщения- отправка сообщений в указанное время
- Напоминания- создание напоминаний для себя или команды
- Повторяющиеся задачи- поддержка периодических уведомлений
- Управление расписанием - просмотр, редактирование и удаление запланированных задач
- Уведомления - оповещения о выполненных задачах

 🔧 Требования

- Mattermost v6.0 или выше
- Go 1.16+ (для сборки из исходников)
- Доступ к файловой системе сервера

 📥 Установка

 Вариант 1: Готовая сборка

1. Скачайте последний релиз со страницы [Releases](https://github.com/lukuchi13/mattermost_mattermost/releases)
2. Загрузите файл `plugin.tar.gz` в панели администратора Mattermost (Системная консоль → Плагины → Загрузить плагин)
3. Включите плагин в списке установленных

 Вариант 2: Сборка из исходников


 Клонирование репозитория
git clone https://github.com/yourusername/mattermost-plugin-scheduler.git
cd mattermost-plugin-scheduler

 Сборка плагина
make dist
 Полученный файл будет в dist/plugin.tar.gz




Настройка:
{
    "scheduler": {
        "enable_periodic_tasks": true,
        "max_scheduled_per_user": 50,
        "timezone": "UTC"
    }
}

 Использование
 
 Основные команды

Создание запланированного сообщения
/schedule create "Текст сообщения" --time "2024-12-31 18:00" --channel town-square

Создание повторяющейся задачи
/schedule repeat "Ежедневная встреча" --cron "0 9 * * *" --channel daily-standup

Просмотр запланированных задач
/schedule list

Удаление задачи
/schedule delete <task-id>


📝 Команды
Команда	          Описание	                          Пример
/schedule create	Создать разовое сообщение	          /schedule create "Встреча" --time "15:00"
/schedule repeat	Создать повторяющуюся задачу	      /schedule repeat "Дэйли" --cron "0 10 * * 1-5"
/schedule list	  Показать все задачи	                /schedule list
/schedule delete	Удалить задачу	                    /schedule delete 123
/schedule help	  Показать справку	                  /schedule help
Параметры команд
--time - время выполнения (формат: YYYY-MM-DD HH:MM или HH:MM)

--cron - CRON-выражение для повторяющихся задач

--channel - канал для отправки (по умолчанию текущий)

--remind - напоминание за N минут до события
